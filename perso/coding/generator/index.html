Everything is so fast on the Internet. Not even fast, instataneous for the most part. This project tried to look at this global trend from a new perpestive. Interaction is about the user performing an action WITH the interface.
For this project I wanted the user to be able to play with the interface with little to no directions on how to use it properly. To me, the essence of interaction is precisely this ability to navigate an interface freely.
For this project I wanted the user to be able to play with the interface with little to no directions on how to use it properly. To me, the essence of interaction is precisely this ability to navigate an interface freely.
The two dots act as the return home button, and the only navigation available on the site. Discrete, yet it plays a more important role in the meaning of the interaction.
The randomly produced poem is taking inspiration from the dada movement. Sentenced from this movement lack punctuation, and can sometime feel too "dense" and not balanced. This generator works the same way, there is no punctuation. The user can sometimes feel lost with the produced poem, the "escape home" button acts as an inside joke : the user will press the punctuation button to get a clearer image and be brought back home.
