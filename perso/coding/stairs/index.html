The goal of this project was to translate a physical space -from the real world- into an interactive web space. I chose to work on a staircase in the lobby of my school in New York.
The second step was to identify three actions performed in the real world, study how these were performed. And translate them into a digital interface.
Each shape in the interface (triangle/circle/rectangle) is a symbol for one of these actions. The user firstly selects one of the shapes, and will discover thorugh riddles which shape is the symbol of which action
The goal of the interface is to have the user "adventure" himself in the different options, and understand what are the real-world actions behind the shapes, as he plays with the interactive product.
On top of the stairs, samples of the currently selected shapes are present. The information that the riddle asks to reveal is hidden behind one of these shapes.
Though the riddle really asks to manipulate elemtns of the interfact to reach the point where the information can be released.
One of the thing I learned most while attending Parsons is problem solving. Back when I worked on this project, my coding skills were not as broad as they are now. I was very set on my design idea, to implement a slider, really giving that interactive control to the user.
